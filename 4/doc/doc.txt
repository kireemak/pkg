Лабораторная работа 4 "Базовые растровые алгоритмы"

Киреенко Макар, 12 группа

Цель работы:

Разработать программу на языке Python с использованием библиотек Matplotlib и Tkinter для
визуализации графических алгоритмов, таких как алгоритм пошагового рисования, DDA-алгоритм,
алгоритм Брезенхэма для прямой и окружности.

Описание проведенной работы:

На основе предоставленного кода был создан класс Program, инкапсулирующий весь функционал
программы. Каждый графический алгоритм (пошаговое рисование, DDA, Брезенхэм для прямой и
окружности) был выделен в отдельный метод класса. Реализовано графическое отображение результатов
алгоритмов с использованием библиотеки Matplotlib и встроенного в Tkinter холста.

Ход работы:

1. Создание класса Program, объединяющего все функции и методы программы.
2. Интеграция графических алгоритмов в методы класса.
3. Инициализация графического интерфейса с использованием библиотеки Tkinter и настройка окна с
графиками с помощью Matplotlib.
4. Реализация метода draw для отображения результатов работы алгоритмов на графиках.
5. Обработка ввода пользователя для задания координат и параметров фигур.

Вывод:

Лабораторная работа значительно улучшила навыки программирования на языке Python и знакомство с
библиотеками Matplotlib и Tkinter. Работа над графическими алгоритмами позволила лучше понять
принципы их работы и применение в компьютерной графике. Программа предоставляет удобный способ
визуализации различных алгоритмов. Полученные навыки программирования и работы с графикой будут
полезными в будущей работе в области разработки программного обеспечения.
