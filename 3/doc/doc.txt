Лабораторная работа 3 "Основы обработки цифровых изображений"

Киреенко Макар, 12 группа

Цель работы:

Разработать программу с использованием библиотек OpenCV для обработки изображений,
предоставляющую функционал открытия, отображения, исходного и обработанного изображения,
а также ряда обработок, таких как адаптивное пороговое преобразование,низкочастотная
фильтрация (сглаживание) и локальная пороговая обработка методами Бернсена и Оцу.

Описание проведенной работы:

На основе предоставленного кода был создан класс ImageProcessor, объединяющий все функции
и методы программы. Программа использует библиотеку OpenCV для работы с изображениями,
библиотеку NumPy для обработки массивов данных, а также Tkinter для создания графического
интерфейса. Программа предоставляет кнопки для открытия нового изображения, отображения
исходного изображения, а также для выполнения адаптивного порогового преобразования,
низкочастотной фильтрации и локальной пороговой обработки двумя методами.

Ход работы:

1. Создание класса ImageProcessor, инкапсулирующего функционал работы с изображениями.
2. Инициализация основных компонентов программы, таких как окно Tkinter, кнопки и элемент
управления изображением.
3. Реализация методов для изменения размера изображения, открытия нового изображения и
отображения его на графическом интерфейсе.
4. Реализация методов для адаптивного порогового преобразования, низкочастотной фильтрации
и локальной пороговой обработки методами Бернсена и Оцу.
5. Создание графического интерфейса с использованием Tkinter и отображение изображений
средствами библиотеки Matplotlib.

Вывод:

Лабораторная работа позволила углубить знания по обработке изображений с использованием
популярных библиотек Python. Работа с библиотеками OpenCV, NumPy и Tkinter дала возможность
освоить методы обработки и анализа изображений, а также создания простого графического
интерфейса. Полученные навыки программирования в области обработки изображений будут полезными
при решении задач в сфере компьютерного зрения и обработки данных.
